Microsoft Azure Services Overview

Microsoft Azure is a cloud computing platform with an ever-expanding set of services to help you build solutions to meet your business goals. This guide provides an overview of key Azure services organized by category.

Compute Services

Azure compute services provide on-demand computing resources for running cloud-based applications.

Azure Virtual Machines

Create Windows or Linux VMs in seconds with full control over configuration. Supports various VM sizes optimized for compute, memory, storage, or GPU workloads.

Azure App Service

Build, deploy, and scale web apps, mobile backends, and RESTful APIs in your preferred programming language without managing infrastructure.

Azure Kubernetes Service (AKS)

Simplify Kubernetes deployment and management with a fully managed service. Scale and orchestrate containerized applications with enterprise-grade security.

Azure Functions

Event-driven, serverless compute service that enables you to run code on-demand without provisioning or managing infrastructure.

Storage Services

Azure storage services offer scalable cloud storage for all types of data.

Azure Blob Storage

Massively scalable object storage for unstructured data like documents, images, videos, backups, and log files with tiered storage options.

Azure Files

Fully managed file shares in the cloud accessible via SMB or REST API, enabling hybrid scenarios and lift-and-shift migrations.

Azure Disk Storage

Block-level storage volumes for Azure VMs with options for ultra-disks, premium SSDs, standard SSDs, and standard HDDs.

Azure Data Lake Storage

Hyperscale repository for big data analytics workloads with hierarchical namespace and HDFS compatibility.

Database Services

Azure offers a variety of database services to meet diverse application needs.

Azure SQL Database

Fully managed relational database service built on SQL Server with built-in intelligence for performance tuning and security.

Azure Cosmos DB

Globally distributed, multi-model database service with support for NoSQL APIs including SQL, MongoDB, Cassandra, Gremlin, and Table.

Azure Database for MySQL

Fully managed MySQL database service with high availability, security, and scalability built in at no additional cost.

Azure Database for PostgreSQL

Fully managed PostgreSQL database with high availability, automated backups, and intelligent performance optimization.

AI & Machine Learning Services

Build intelligent applications with Azure's comprehensive set of AI and ML services.

Azure Machine Learning

End-to-end platform for preparing data, training, testing, deploying, managing, and tracking ML models with support for Python and R.

Azure Cognitive Services

Pre-built AI APIs for vision, speech, language, decision, and search integration to build intelligent applications quickly.

Azure OpenAI Service

Enterprise-ready API access to OpenAI's GPT models with the security and compliance features of the Azure platform.

Azure Bot Service

Develop intelligent, enterprise-grade bots that interact naturally with users across multiple channels and devices.

Networking Services

Connect your on-premises and cloud resources with Azure's comprehensive networking services.

Azure Virtual Network

Provision private networks, optionally connect to on-premises datacenters, and isolate Azure resources with secure network boundaries.

Azure Load Balancer

Distribute network traffic across multiple servers to ensure high availability and reliability for your applications.

Azure VPN Gateway

Connect on-premises networks to Azure through site-to-site VPNs, or create point-to-site connections for remote access.

Azure Content Delivery Network

Deliver high-bandwidth content globally with low latency through a distributed network of edge servers.

Security Services

Protect your applications, data, and infrastructure with Azure's comprehensive security services.

Azure Active Directory

Identity and access management service providing single sign-on, multi-factor authentication, and conditional access to applications.

Azure Security Center

Unified security management system that strengthens the security posture of your data centers and provides threat protection.

Azure Key Vault

Safeguard cryptographic keys, certificates, and secrets used by cloud applications and services with hardware security module backing.

Azure Sentinel

Cloud-native SIEM and SOAR solution providing intelligent security analytics across your enterprise.

Integration Services

Connect applications, data, and processes across on-premises and cloud environments.

Azure Logic Apps

Automate workflows and business processes by integrating apps, data, systems, and services across enterprises or organizations.

Azure API Management

Publish, secure, transform, maintain, and monitor APIs across clouds and on-premises environments.

Azure Service Bus

Reliable cloud messaging service between applications and services with guaranteed message delivery and transaction support.

Azure Event Grid

Fully managed event routing service that enables event-based architectures with near-real-time event delivery.

DevOps Services

Build, test, and deploy applications with integrated DevOps tools.

Azure DevOps

Development collaboration tools including pipelines, Git repositories, Kanban boards, and automated testing tools to build, test, and deploy.

GitHub Actions for Azure

Automate software workflows directly from your GitHub repository with built-in CI/CD that works with Azure services.

Azure Monitor

Comprehensive solution for collecting, analyzing, and acting on telemetry from cloud and on-premises environments.

Azure DevTest Labs

Quickly create environments for development and testing with reusable templates and artifacts.
